## 🚀 Principais Features

### 1️⃣ Chrome Built-in AI Integration (5 APIs)

✅ **Prompt API** (languageModel) - Code explanation & analysis
✅ **Writer API** - Story generation & documentation
✅ **Rewriter API** - Code refactoring & optimization
✅ **Summarizer API** - Quick summaries & error condensation
✅ **Translator API** - Multi-language support (future)

### 2️⃣ Análise Multi-Dimensional (5 Dimensões)

- **Readability** (25%) - Variable names, comments, structure
- **Efficiency** (20%) - Algorithmic complexity, performance
- **Best Practices** (25%) - Coding standards, design patterns
- **Security** (15%) - OWASP Top 10, vulnerability detection
- **Maintainability** (15%) - Code duplication, function length

**Output:** Score 1-10, Grade A+ to F, Acionable recommendations

### 3️⃣ Security Analysis (OWASP Top 10)

- XSS, SQL Injection, CSRF detection
- Hardcoded secrets identification
- CWE mapping com remediação
- Severity classification (CRITICAL/HIGH/MEDIUM/LOW)

### 4️⃣ Performance Advisor

- Algorithmic complexity detection (O(n²), O(n³))
- Memory leak identification
- DOM manipulation issues
- **Speedup estimates** (ex: "3.5x faster with Map")

### 5️⃣ Gamificação Completa

- Sistema de XP e níveis (1-100)
- 20+ badges em 6 categorias
- Daily challenges & streak tracking
- Leaderboards (global, team, weekly)

### 6️⃣ Adaptive Learning Engine

- Skill level tracking (beginner → expert)
- Learning style detection (visual, auditory, etc)
- Personalized learning paths
- Knowledge gap identification

## 📦 Instalação

### Pré-requisitos

- Chrome 130+ ([como verificar](chrome://version/))
- Windows 10+, macOS 10.15+, ou Linux

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/v1olegrace/DevMentorIA.git
   cd DevMentorIA
   ```

2. **Carregar extensão no Chrome:**
   - Abra `chrome://extensions/`
   - Ative "Modo do desenvolvedor"
   - Clique "Carregar sem compactação"
   - Selecione a pasta `devmentor-ai/`

3. **Verificar instalação:**
   - Ícone deve aparecer na toolbar
   - Clique → "Test Connection"
   - Deve mostrar "✅ All APIs available"

4. **(Opcional) Backend Setup:**
   - Veja [QUICK_START.md](./QUICK_START.md) para setup completo

## 🎮 Como Usar

### Uso Básico (30 segundos)

1. Vá para qualquer site com código (GitHub, StackOverflow, etc)
2. Selecione um trecho de código
3. Botão direito → "DevMentor AI" → escolha ação
4. Veja resultados em overlay elegante

### Features Avançadas

#### Code Storytelling
```
1. Selecione código

3. Escolha tema (Adventure, Mystery, etc)
4. Leia história + visualizações
5. Faça quiz interativo
6. Exporte para HTML/Markdown
```

#### Comprehensive Analysis
```

```
